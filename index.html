GitHub-ல் உங்கள் வெப்சைட்டை வெளியிடுவது எப்படி? வணக்கம்! இந்த வழிகாட்டி மூலம், உங்களுக்குக் கொடுக்கப்பட்ட index.html கோடைப் பயன்படுத்தி, GitHub Pages-ல் உங்கள் வெப்சைட்டை இலவசமாக வெளியிடுவது எப்படி என்று பார்க்கலாம். படிப்படியான வழிமுறைகள் (Step-by-Step): படி 1: GitHub கணக்கு * உங்களுக்கு GitHub (www.github.com) கணக்கு இல்லையென்றால், புதிதாக ஒன்றை உருவாக்கிக்கொள்ளுங்கள். படி 2: புதிய Repository-ஐ உருவாக்குதல் * GitHub-ல் லாகின் செய்த பிறகு, உங்கள் முகப்புப் பக்கத்தில் (homepage) New என்ற பச்சை நிற பட்டனை கிளிக் செய்யவும் (அல்லது மேல் வலது மூலையில் உள்ள + ஐகானை கிளிக் செய்து New repository என்பதைத் தேர்ந்தெடுக்கவும்). * முக்கியமான படி: Repository name (களஞ்சியத்தின் பெயர்) என்பதற்கு, பின்வருமாறு சரியாக டைப் செய்யவும்: your-username.github.io * (இங்கே your-username என்பதற்குப் பதிலாக உங்கள் உண்மையான GitHub யூசர்நேமைப் பயன்படுத்தவும். உதாரணம்: உங்கள் யூசர்நேம் actingdriver என்றால், actingdriver.github.io என்று பெயரிட வேண்டும்). * Public என்பதைத் தேர்ந்தெடுத்து, Create repository பட்டனை கிளிக் செய்யவும். படி 3: index.html பைலை உருவாக்குதல் * உங்கள் புதிய repository-ஐ உருவாக்கியதும், அந்தப் பக்கத்தில் Add file என்பதைக் கிளிக் செய்து, பின்னர் Create new file என்பதைத் தேர்ந்தெடுக்கவும். * பைலின் பெயருக்கான கட்டத்தில் (name your file...), index.html என டைப் செய்யவும். (இது மிகவும் முக்கியம், பெயர் சரியாக index.html என்று இருக்க வேண்டும்). * அதற்குக் கீழே உள்ள பெரிய டெக்ஸ்ட் பாக்ஸில் (text box), நான் உங்களுக்கு வழங்கிய index.html பைலில் உள்ள முழு கோடையும் ( என்பதில் தொடங்கி என முடியும் வரை) காப்பி (Copy) செய்து, இங்கே பேஸ்ட் (Paste) செய்யவும். படி 4: பைலை சேமித்தல் (Commit) * கோடை பேஸ்ட் செய்த பிறகு, பக்கத்தின் கீழே Commit new file என்ற தலைப்பு இருக்கும். * அங்குள்ள பச்சை நிற Commit new file பட்டனை கிளிக் செய்யவும். இது உங்கள் பைலை சேமிக்கும். படி 5: உங்கள் வெப்சைட்டைப் பாருங்கள்! அவ்வளவுதான்! உங்கள் வெப்சைட் இப்போது ஆன்லைனில் உள்ளது. சரியாக 5 முதல் 10 நிமிடங்கள் வரை காத்திருக்கவும். அதன் பிறகு, உங்கள் பிரவுசரில் (Browser) https://your-username.github.io என்று டைப் செய்து சென்றால் (உதாரணம்: https://actingdriver.github.io), உங்கள் புதிய வெப்சைட்டை நீங்கள் பார்க்கலாம்! அடுத்தகட்ட நடவடிக்கைகள்: * வீடியோக்களை மாற்றுதல்: index.html கோடில், உங்கள் சொந்த YouTube வீடியோக்களின் "embed" லிங்கை மாற்ற மறவாதீர்கள். * படங்களை மாற்றுதல்: "காட்சிகள்" (Gallery) பகுதியில், உங்கள் சொந்த படங்களுக்கான URL-களைப் பயன்படுத்தவும். * இணைப்புகளை மாற்றுதல் (Links): உங்கள் Facebook, Instagram மற்றும் Google My Business இணைப்புகளைச் சரியாக அப்டேட் செய்யவும்.
